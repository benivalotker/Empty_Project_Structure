## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirement package.

For unix system use the MakeFile:

```bash
make -f Makefile
```

For Windows system use the requirements.txt: 
```bash
pip install -r requirements.txt
```
